Σκοπός της εργασίας είναι η ανάπτυξη ενός ολοκληρωμένου εργαλείου αυτοματοποίησης σχεδίασης που υλοποιεί τα βασικά βήματα της λογικής σύνθεσης.  Στόχος είναι η κατανόηση της διαδικασίας μετάβασης από μια αφηρημένη μαθηματική περιγραφή (πίνακας αληθείας) σε φυσικό υλικό (πύλες).  Το εργαλείο επιτρέπει την εκτίμηση της απόδοσης του κυκλώματος υπολογίζοντας μεγέθη όπως το εμβαδόν και την καθυστέρηση.  Δίνεται έμφαση στη σημασία της επιλογής των κατάλληλων πυλών από την δοσμένη βιβλιοθήκη
